# PullRequests

Handle pull requests

## Cmdlets

| Cmdlet | Description |
|--------|-------------|
| [Close-GithubPullRequest](PullRequests/Close-GithubPullRequest.md) | Closes a GitHub pull request. |
| [Get-GithubPullRequest](PullRequests/Get-GithubPullRequest.md) | Gets pull requests from a GitHub repository. |
| [Get-GithubPullRequestComment](PullRequests/Get-GithubPullRequestComment.md) | Gets comments on a GitHub pull request. |
| [Merge-GithubPullRequest](PullRequests/Merge-GithubPullRequest.md) | Merges a GitHub pull request. |
| [New-GithubPullRequest](PullRequests/New-GithubPullRequest.md) | Creates a new pull request in a GitHub repository. |
| [New-GithubPullRequestComment](PullRequests/New-GithubPullRequestComment.md) | Adds a comment to a GitHub pull request. |
| [Update-GithubPullRequest](PullRequests/Update-GithubPullRequest.md) | Updates an existing pull request. |
