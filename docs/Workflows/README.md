# Workflows

GitHub Actions workflows

## Cmdlets

| Cmdlet | Description |
|--------|-------------|
| [Get-GithubWorkflow](Workflows/Get-GithubWorkflow.md) | Gets GitHub Actions workflows from a repository. |
| [Get-GithubWorkflowJob](Workflows/Get-GithubWorkflowJob.md) | Gets jobs from a GitHub Actions workflow run. |
| [Get-GithubWorkflowRun](Workflows/Get-GithubWorkflowRun.md) | Gets GitHub Actions workflow runs from a repository. |
| [Get-GithubWorkflowRunLog](Workflows/Get-GithubWorkflowRunLog.md) | Gets the log output for a GitHub Actions workflow job. |
| [Start-GithubWorkflowRun](Workflows/Start-GithubWorkflowRun.md) | Triggers a GitHub Actions workflow run. |
