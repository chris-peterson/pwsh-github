# New-GithubRepository

## SYNOPSIS

Creates a new GitHub repository.

## SYNTAX

### __AllParameterSets

```
New-GithubRepository [-Name] <string> [-Organization <string>] [-Description <string>]
 [-Visibility <string>] [-Homepage <string>] [-HasIssues] [-HasWiki] [-AutoInit] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Creates a new GitHub repository for the authenticated user or within an organization.

## EXAMPLES

### Example 1

New-GithubRepository 'my-repo' -Visibility public
