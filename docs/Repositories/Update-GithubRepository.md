# Update-GithubRepository

## SYNOPSIS

Updates a GitHub repository.

## SYNTAX

### __AllParameterSets

```
Update-GithubRepository [[-RepositoryId] <string>] [[-Name] <string>] [[-Description] <string>]
 [[-Visibility] <string>] [[-Homepage] <string>] [[-DefaultBranch] <string>] [-Archived] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Updates properties of an existing GitHub repository such as name, description, visibility, homepage, default branch, and archived status.

## EXAMPLES

### Example 1

Update-GithubRepository -Description 'Updated description'
