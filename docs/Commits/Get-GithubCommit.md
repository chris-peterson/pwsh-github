# Get-GithubCommit

## SYNOPSIS

Gets commits from a GitHub repository.

## SYNTAX

### __AllParameterSets

```
Get-GithubCommit [[-Sha] <string>] [-RepositoryId <string>] [-Branch <string>] [-Author <string>]
 [-Since <string>] [-Until <string>] [-Path <string>] [-MaxPages <uint>] [-All] [-Select <string>]
 [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Gets commits from a GitHub repository. When a SHA is specified, retrieves detailed information for that commit. Otherwise, lists commits with optional filtering by branch, author, date range, and file path.

## EXAMPLES

### Example 1

Get-GithubCommit
Get-GithubCommit -Branch main -Author octocat
