# New-GithubMilestone

## SYNOPSIS

Creates a new milestone in a GitHub repository.

## SYNTAX

### __AllParameterSets

```
New-GithubMilestone [[-RepositoryId] <string>] [-Title] <string> [[-Description] <string>]
 [[-DueOn] <string>] [[-State] <string>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Creates a new milestone in a GitHub repository with the specified title and optional description, due date, and state.

## EXAMPLES

### Example 1

New-GithubMilestone -Title 'v1.0'
