# Update-GithubIssue

## SYNOPSIS

Updates an existing GitHub issue.

## SYNTAX

### __AllParameterSets

```
Update-GithubIssue [-IssueId] <int> [-RepositoryId <string>] [-Title <string>]
 [-Description <string>] [-State <string>] [-Assignees <string[]>] [-Labels <string[]>]
 [-MilestoneNumber <int>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Updates properties of an existing GitHub issue such as title, description, state, assignees, labels, and milestone.

## EXAMPLES

### Example 1

Update-GithubIssue 42 -State closed
