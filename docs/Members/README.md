# Members

Organization and repository members

## Cmdlets

| Cmdlet | Description |
|--------|-------------|
| [Add-GithubOrganizationMember](Members/Add-GithubOrganizationMember.md) | Adds or updates a user's membership in a GitHub organization. |
| [Add-GithubRepositoryCollaborator](Members/Add-GithubRepositoryCollaborator.md) | Adds a collaborator to a GitHub repository. |
| [Get-GithubOrganizationMember](Members/Get-GithubOrganizationMember.md) | Gets members of a GitHub organization. |
| [Get-GithubRepositoryCollaborator](Members/Get-GithubRepositoryCollaborator.md) | Gets collaborators of a GitHub repository. |
| [Remove-GithubOrganizationMember](Members/Remove-GithubOrganizationMember.md) | Removes a member from a GitHub organization. |
| [Remove-GithubRepositoryCollaborator](Members/Remove-GithubRepositoryCollaborator.md) | Removes a collaborator from a GitHub repository. |
