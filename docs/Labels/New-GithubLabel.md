# New-GithubLabel

## SYNOPSIS

Creates a new label in a GitHub repository.

## SYNTAX

### __AllParameterSets

```
New-GithubLabel [[-RepositoryId] <string>] [-Name] <string> [-Color] <string>
 [[-Description] <string>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Creates a new label in a GitHub repository with the specified name, color, and optional description.

## EXAMPLES

### Example 1

New-GithubLabel -Name 'bug' -Color 'd73a4a'
