# Update-GithubLabel

## SYNOPSIS

Updates an existing label in a GitHub repository.

## SYNTAX

### __AllParameterSets

```
Update-GithubLabel [-Name] <string> [-RepositoryId <string>] [-NewName <string>] [-Color <string>]
 [-Description <string>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Updates properties of an existing label such as name, color, and description.

## EXAMPLES

### Example 1

Update-GithubLabel 'bug' -Color 'ff0000'
