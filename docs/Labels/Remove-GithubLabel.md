# Remove-GithubLabel

## SYNOPSIS

Deletes a label from a GitHub repository.

## SYNTAX

### __AllParameterSets

```
Remove-GithubLabel [-Name] <string> [-RepositoryId <string>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Deletes a label from a GitHub repository.

## EXAMPLES

### Example 1

Remove-GithubLabel 'wontfix'
